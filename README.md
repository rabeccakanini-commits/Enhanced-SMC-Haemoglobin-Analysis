# Enhanced SMC Haemoglobin Analysis

This repository contains the analysis conducted for the Advanced Data Analysis project at the African Institute for Mathematical Sciences (AIMS).

## Project Title
Impact of Enhanced Seasonal Malaria Chemoprevention (eSMC) on Haemoglobin Trajectories in Children

## Description
This project evaluates the effect of Enhanced Seasonal Malaria Chemoprevention on haemoglobin levels in children over a 6-month randomized controlled trial. The analysis focuses on modelling haemoglobin trajectories using linear mixed effects models.

## Methods
The analysis includes:
- Exploratory data analysis
- Linear mixed effects modelling
- Model comparison and diagnostics
- Longitudinal trajectory interpretation

## Tools
- R
- lme4
- ggplot2
